Movement- and Traffic-based User Identification in Commercial Virtual Reality Applications: Threats and Opportunities
Sara Baldoni, Salim Benhamadi, Federico Chiariotti, Michele Zorzi, Federica Battisti
IEEE Conference on Virtual Reality and 3D User Interfaces (IEEE VR), 2025
With the unprecedented diffusion of virtual reality, the number of application scenarios is continuously growing. As commercial and gaming applications become pervasive, the need for secure and convenient identification of users, often overlooked by research in immersive media, is becoming increasingly pressing. This work investigates user identification through movement patterns and data traffic traces in commercial VR games, analyzing both opportunities for content customization and privacy implications.

Automated Semiconductor Test Data Analysis Using a Hybrid Deep Learning Pipeline
Salim Benhamadi, Massimo Pierobon, Giorgia Cassetta, Sabrina Santirosi
5th European Automotive Reliability, Test and Safety Workshop (eARTS), 2025
Manual analysis of test data plots in semiconductor manufacturing creates a significant bottleneck in production optimization, with product engineers dedicating extensive hours to reviewing patterns, anomalies, and making assessments. This study presents an automated approach based on computer vision and machine learning to streamline this process. We developed a multi-stage pipeline that detects and extracts plots from various document formats using Faster R-CNN, generates feature embeddings via Vision Transformer (ViT-B/32), and employs Affinity Propagation clustering alongside Isolation Forest for anomaly detection.
